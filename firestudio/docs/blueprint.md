# **App Name**: AgriAssist

## Core Features:

- Smart Crop Suggestions: Crop Recommendation Tool: Based on user's soil type, weather and location (which must be entered manually), provide LLM-powered crop recommendations. As a tool, the LLM can assess user inputs to determine if there's enough information to perform this function.
- Crop Info Cards: Simple, card-based layout for crop information display on all screen sizes.
- Multi-Lingual Support: Support for English, Hindi, Tamil, and Telugu.
- Language Selection: Easy language toggle in settings menu
- Weather Integration: OpenWeather API to show weather forecast.
- 7-Day Weather Display: Display seven day weather forecast
- Disease Diagnosis: Farmers upload leaf images, and a CNN model (or Gemini Vision) detects diseases, showing solutions with both text and voice output. Store image and results in Firebase Cloud Storage and Firestore.
- Government Policies: Display relevant Indian government policies related to farming and agriculture.
- Speaker: Enable voice output (text-to-speech) for multilingual support.

## Style Guidelines:

- Primary color: Forest green (#3F704F) to represent agriculture and growth.
- Background color: Light beige (#F5F5DC) to provide a natural and earthy feel.
- Accent color: Warm yellow (#FFC300) to highlight important information and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear and accessible text.
- Use clear, high-contrast icons for easy navigation and quick understanding.
- Mobile-first design with large, clear buttons and a simplified layout for ease of use on smaller screens.
